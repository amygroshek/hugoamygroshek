I really like the Really Simple Twitter Feed Widget for Wordpress, but retweets and things with a lot of links don't display well out of the box:

Turns out WP has a default style for links in sidebar widgets:

.widget-area li a {
    padding: 2px 0;
    display: block;
}

Now, for tweets, where we want a compact visual presentation, display: block; is just not the right way to go. I added the following simple tweak to styles.css:

.really_simple_twitter_widget li a {
    display:inline;
}

All fixed:

Update

I added an additional fix to prevent short tweets from being wrapped by the subsequent tweet:

.really_simple_twitter_widget li {
    margin:0.5em;
    display:inline-block;
}
